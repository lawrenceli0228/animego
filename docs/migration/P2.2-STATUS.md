## 实现层

### 新增包

| 包 | LoC | Coverage | 说明 |
|---|---|---|---|
| `internal/jwtx` | ~660 | **92.4%** | 双 HS256 密钥 + 33 test |
| `internal/auth` | ~2080 | **87.9%** | 5 handler + ratelimiter + cookies |

### internal/jwtx

- `Signer` 双密钥(access 15m + refresh 7d,JWT_SECRET / JWT_REFRESH_SECRET 分开,leak 一个不能伪造另一个)
- `AccessClaims{UserID, Username, Role *string omitempty, exp/iat}` — Role nil 时 JSON 不出现(跟 Express conditional spread 一致)
- `RefreshClaims{UserID, exp/iat}` — userId only
- `HashPassword` + `ComparePassword` cost=10(跟 Express bcryptjs 互通,Mongo migration 老 hash 直接验)
- `RequireAuth` middleware dual-accept:`Authorization: Bearer <token>` OR `accessToken` cookie。Header 优先。401 envelope byte-exact 中文(`需要登录` / `无效的 token`)
- alg-confusion defense:`*jwt.SigningMethodHMAC` 强制 assert,挡 `alg=none` + `alg=RS256` 攻击
- `ClaimsFrom(ctx)` accessor + package-private context key(外部不能伪造 key)

### internal/auth

- `SafeUser` 投影脱敏 password / refreshToken / resetPasswordToken / resetPasswordExpires 4 字段 — `ToSafeUser(dbgen.User)` 单一转换点
- 5 handler 全用 `errgroup` / `validator.v10` / pgconn.PgError race-detect
- Register `pgconn 23505` SQLSTATE → 400 DUPLICATE(防 pre-check race condition)
- Login 错邮箱 + 错密码同 `INVALID_CREDENTIALS` + `邮箱或密码错误` message(防 enumeration)
- Refresh 双 check:`signer.VerifyRefresh` + DB.`refresh_token` 必须等于 cookie(rotation defense)
- Logout `UpdateUserRefreshToken(id, nil)` 清 DB + `ClearRefreshCookie` 清客户端
- Cookies:`httpOnly` + `SameSite=strict`(dev)/`none`(prod)+ `path=/api`(限 CSRF 面)+ `maxAge=7d`
- Rate limiter:fixed-window 10 req / 15 min per IP,sweeper goroutine 每 5min GC,`Stop()` `sync.Once` 守

### Validator → 中文消息映射

| Validator 错 | 中文消息(byte-exact Express) |
|---|---|
| username min/max | `用户名需 3-50 个字符` |
| email | `邮箱格式不正确` |
| password min | `密码至少 6 位` |
| password required(login) | `密码不能为空` |
| unmapped | `参数错误`(默认 fallback,不泄露 validator 英文) |

## SQL 层

`internal/db/queries/users.sql` 加 8 个 sqlc method:

```text
CreateUser(username, email, hashed_pw) → User
GetUserByEmail(email) → User
GetUserByUsername(username) → User
GetUserByID(uuid) → User
UpdateUserRefreshToken(id, *token) → ()
SetResetPasswordToken(id, *token, expires) → ()
GetUserByResetToken(token) → User       ← P2.2.4 用
ResetUserPassword(id, hashed_pw) → ()   ← P2.2.4 用
```

前 5 个被 P2.2.2 handlers 消费,后 3 个等 P2.2.4 forgot/reset 接入。

## Config 加项

```text
JWTSecret               (HS256 access secret,P2.2 起必填)
JWTRefreshSecret        (HS256 refresh secret,跟 access 分开)
JWTExpiresIn            (default 15m,支持 Express "15m" shorthand)
JWTRefreshExpiresIn     (default 7d,支持 "7d" shorthand 也支持 Go duration)
```

`envDuration` helper 处理 Express 的 `7d` 简写(Go `time.ParseDuration` 不认 d)。
