## Bangumi 富化 pipeline

```
+----------+    chain    +----------+    chain    +----------+
|    V1    | ----------> |    V2    | ----------> |    V3    |
| (live)   | hit + bgmID | (live)   | (P2.1.8)    | (stub)   |
+----------+             +----------+             +----------+
   ^                                                   ^
   |                                                   |
   +------- enqueue source -------+                    +-- 待 P2.1.8
           /search post-upsert
           /schedule unenriched
           boot orphan scan (bangumi_version=0)
```

**V1 worker** (`internal/queue/bangumi_v1.go`): Bangumi search by titleNative → 精确匹配设 title_chinese / bgm_id / bangumi_version=1 / chain V2

**V2 worker** (`internal/queue/bangumi_v2.go`): Bangumi Subject + Characters → 写 bangumi_score / votes / 每 character UPDATE name_cn + voice_actor_cn + voice_actor_image_url / bangumi_version=2

**V3 stub**: 待 P2.1.8 替换。功能是 heal-CN(V2 之后 title_chinese 还 NULL 的行重 Subject 一次)。
